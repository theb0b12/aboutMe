Ergonomic Mechanical Keyboards

A deep dive into custom keyboard design using Ergogen, a YAML based generator for building split, column staggered keyboardss. This project started as a personal interest in the mechanical keyboard hobby and became a study in how small geometry decisions — key angle, thumb cluster placement, column offset — compound into dramatically different typing experiences. The repository contains Ergogen configuration files, generated KiCad footprints, and firmware mapping.

Blades of Dawn

A SFML-based platformer built in C++ as the final project for CSCI 200 at Colorado School of Mines. Blades of Dawn is a fast paced, if not rage game implementing object oriented design patterns — inheritance hierarchies for character classes, and a simple™ state machine for game flow. It was the my largest software project undertaken at the time and required careful planning of class architecture and overall project design.

b0bqol — Fabric Minecraft Mod

A client-side Fabric mod written in Java for use on private and public Minecraft servers. The mod's scope is deliberately narrow: quality-of-life improvements focused on piston door and redstone building , addressing specific annoyances in vanilla gameplay. My first real mod published on a open source repository.
